Control system projects

This page presents a selection of control systems projects.

# Project Title Description Document
1 Modeling and control of a four-cable suspended cable-driven robot State-space modeling and Linear Quadratic Regulator (LQR) control of a four-cable suspended cable-driven robot for precise position regulation.
2 Modeling and state feedback control of a mini UAV State-space modeling and state feedback control of a mini UAV for attitude and position stabilization.
3 State feedback control of a DC motor Design and tuning of a state feedback controller for speed regulation of a DC motor.
4 State-space control of an inverted pendulum Modeling and control of an inverted pendulum using LQR and pole-placement techniques.
5 Trajectory tracking for a mobile robot Implementation of a path-following algorithm for a differential-drive robot using feedback control.
6 Robust control of a 2-DOF manipulator Comparison of PID and model-based controllers under parameter variations and disturbances.
7 Model Predictive Control (MPC) for Temperature Regulation Design of a predictive controller for temperature regulation in a thermal chamber system.
